template<size_t N, typename = typename std::enable_if_t<(N >= 4 && N <= 32)>>
constexpr uinteger_t< N > saturation::mulu ( uinteger_t< N > const  x,
uinteger_t< N > const  y 
)
constexpr

Computes the value of x × y.

If the result overflows — that is, the correct answer is too large to be representable with an unsigned integer of N bits — the returned value is \( 2^N-1 \).

Template Parameters
NThe number of bits for the unsigned arguments and result. May be in the range \( [4, 64] \).
Parameters
xThe first value to be multiplied.
yThe second value to be multiplied.
Returns
x × y. If the result would be too large, \( 2^N-1 \) (saturation::limits<N>::max()).
