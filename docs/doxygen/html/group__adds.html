Adds two signed values each N bits wide.

Returns \( 2^{N-1}-1 \) or \( -2^{N-1} \) if the result cannot be represented in N bits.

Template Parameters
NThe number of bits for the unsigned arguments and result. May be in the range \( [4, 64] \).
Parameters
xThe first of the two values to be added.
yThe second of the two values to be added.
Returns
x + y or \( 2^{N-1}-1 \) if the result is positive but cannot be represented in N bits; \( -2{N-1} \) if the result is negative but cannot be represented in N bits.
