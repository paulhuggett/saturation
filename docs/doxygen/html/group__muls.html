constexpr

Computes the signed result of multiplying x by y.

Template Parameters
NThe number of bits for the twos complement arguments and result. May be in the range \( [4, 32] \).
Parameters
xThe first value to be multiplied.
yThe second value to be multiplied.
Returns
x × y. If the result would be too large and positive, \( 2^{N-1}-1 \) (saturation::limits<N>::max()); if the result would be too large and negative, \( -2^{N-1} \) (saturation::limits<N>::min()).
