Detailed Description

template<size_t N, typename = typename std::enable_if_t<(N <= 64)>>
struct saturation::mask< N, typename >

Equivalent to (T{1}<<N)-T{1} where T is an unsigned integral type.

Avoids the risk of overflow if N is equal to the number of bits in T. Yields 0 if N is 0.
