)
constexpr

Computes the signed result of x / y.

Note
Twos complement signed division can overflow because \( \frac{-2^{N-1}}{-1} = 2^{N-1} \).
Template Parameters
NThe number of bits for the twos complement arguments and result. May be in the range \( [4, 64] \).
Parameters
xThe signed dividend.
yThe signed divisor.
Returns
x / y. If the result would be too large and positive, \( 2^{N-1}-1 \) (saturation::limits<N>::max()); if the result would be too large and negative, \( -2^{N-1} \) (saturation::limits<N>::min()).
