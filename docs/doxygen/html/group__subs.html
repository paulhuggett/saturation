sinteger_t< N > const  y 
)
constexpr

Computes the signed result of x - y.

If the result overflows — that is, the result is either too large or too small to be representable with a twos-complement integer of N bits — the result is \( 2^{N-1}-1 \) or \( -2^{N-1} \) respectively.

Template Parameters
NThe number of bits for the twos complement arguments and result. May be in the range \( [4, 64] \).
Parameters
xThe value from which y is deducted.
yThe value deducted from x.
Returns
x - y. If the result would be too small, \( -2^{N-1} \) (saturation::limits<N>::min()); if the result would be too large, \( 2^{N-1}-1 \) (saturation::limits<N>::max()).
